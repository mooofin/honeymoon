v0.5 - Config Mixin, Undo/Redo & Working Settings

commit 6815087

  • Extracted Config from a nested struct into a proper CRTP-compatible mixin (Editor : private Config, private UndoHistory<>)
  • Settings persist to ~/.config/honeymoon/config.moon with XDG-aware path resolution and automatic directory creation
  • Full key set (tab_width, show_line_numbers, syntax_highlighting, color_theme, scroll_offset, font_size) serializes as clean key value pairs with # comment support
  • UndoHistory<> mixin brings snapshot-based undo/redo with contiguous-typing coalescing (typing gate so hello is one undo step)
  • Every buffer-mutating path snapshots before touching state (cut, yank, kill_line, kill_word, transpose_chars/words, indent/dedent, delete_forward/backward, newline)
  • Undo/redo applies via delete_range(0, size()) + insert_string (O(n) restore, O(1) snapshot)
  • Bound C-/ → undo, M-/ → redo
  • Settings screen actually works: arrow keys navigate, Enter toggles booleans, Tab Width cycles 2→4→8, and every change auto-saves
