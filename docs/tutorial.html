Navigation

Honeymoon supports both standard arrow keys and Emacs-style bindings.

  • Ctrl + P : Move Up
  • Ctrl + N : Move Down
  • Ctrl + B : Move Left
  • Ctrl + F : Move Right

Editing

Basic text entry works as expected. Here are some power commands:

  • Enter : Insert new line
  • Backspace / Ctrl + H : Delete character backwards
  • Del : Delete character forwards

Clipboard & Selection

1. Set a Mark: Ctrl + Space

2. Select: Move your cursor.

3. Cut/Copy:

  • Ctrl + W : Cut
  • Esc then w : Copy

4. Paste: Ctrl + Y

File Operations

  • Ctrl + X then Ctrl + S : Save
  • Ctrl + X then Ctrl + C : Quit
